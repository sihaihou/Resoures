元素选择符:
----通配符:*{ }
----类选择符：点+类名称{ }
----id选择符：#id的名称{ }
----类型选择符(标签)：标签{ }

关系选择符:
---1,子元素选择器：父亲>儿子
---2,兄弟选择器：自己 ~ 兄弟
---3,相邻选择器：E + F
---4,包含选择器E F
属性选择符：
----1,当前元素[属性]{ }
----2,当前元素[属性="属性值"]{ }
伪装选择符:
----1,link
----2,visited
----3,hover
----4.active
 伪对象(元素)选择符
----1,元素：before{ }
----2,元素：after{ }
选择器的优先级：
---- ！import(无穷大) > 内联(1000) > ID(100) > 类(10) > 标签|伪类|属性选择器(1) > 伪对象 > 继承 > 通配符
