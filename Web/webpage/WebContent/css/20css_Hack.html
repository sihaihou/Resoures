css hack方法一：
--条件注释法
----1，只在IE下生效：
------《!--[if IE]》
----这段文字只在IE浏览器显示
-----《![endif]--》
----2,只在IE6下生效：
------《!--[if IE 6]》
----这段文字只在IE6浏览器显示
------《![endif]--》
----3:只在IE6以上版本生效：
------《!--[if gte IE 6]》
----这段文字只在IE6以上版本浏览器显示
------《![endif]--》
--类内属性前缀法:
----1，属性前缀法是在css样式属性名前加上一些只有特定浏览器才能识别的Hack前缀已达到预期的页面展现效果。
----2，在标准模式中：
------"-" ：下划线IE6识别
------"*" ：星号IE7识别
------"\9" ：IE6、7、8、9、10都能识别
------"\0" ：IE8、9、10都识别
------"\9\0" ：只对IE9、10都识别
--选择器前缀法：
----1，选择器前缀法是针对一些页面表现不一致或者需要特殊对待的浏览器，在css选择器前加上一些只有某些特定浏览器才能识别的前缀进行Hack。并不很常用。
----2，目前常见的是：
------*html* ：前缀只对IE6生效。
------*+html*+ ：前缀只对IE7生效。
