背景：
----1、background-attachment 背景图像是否固定或者随着页面的其余部分滚动
----2、background-color 设置元素的背景颜色
----3、background-image 把图片设置为背景 使用url("图片")
----4、background-position 设置背景图片的起始位置
----5、background-repeat 设置背景图片是否如何重复


--颜色：
一、颜色值：
---- 1、颜色名称
---- 2、十六进制数
---- 3、RGB
---- 4、RGBA
---- 5、透明色
二、属性：
----1、opactiy：\number\ 设置对象的不透明度；值为1：不透明 值为0：完全透明


--边框：
----1、border-width：边框宽度
----2、border-style：边框样式
----3、border-color：边框颜色
----4、border-radius：设置圆角边框
----5、box-shadow：设置对象阴影
----6、border-image：边框背景图片
